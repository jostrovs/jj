Toggleable jQuery plugin

Tarjoaa helpon tavan näyttää/piilottaa div-elementtejä klikkaamalla.

Voidaan sitoa html:ssä joko piilotettavaan diviin itseensä tai sitä edeltävään elementtiin.

Sidonta voidaan tehdä seuraavilla tavoilla:

  1. Asetetaan piilotettavalle div-elementille luokka jj-toggle
  2. Asetetaan piilotettavaa div-elementtiä edeltävälle elementille (otsikolle) elementille luokka jj-toggle-next
  3. Kutsutaan jQueryllä piilotettavalle div-elementille plugin-metodia jjToggleable
  4. Kutsutaan jQueryllä otsikkoelementille plugin-metodia jjNextToggleable

jQuery-plugin-metodit hyväksyvät options-objektin, jolla on seuraavat asetukset:

  • hideOnStart: true Piilotetaanko div oletuksena
  • addToggleElement: true Lisätäänkö toggle-elementti: +
  • useHeaderElement: true Reagoiko otsikkoelementti klikkaukseen
Jos toggle-elementtiä ei lisätä, käytetään otsikkoa aina (muutoin ei olisi mitään tapaa saada piilotettua elementtiä takaisin).

Käyttö:
jQyery täytyy tietysti löytyä, ja sen jälkeen jj.js. Samoin jj.css oletetaan löytyvän, mutta siinä on ainoastaan toggle-elementtien tyylittely.

Esimerkki

div-elementtiin sitominen:

<span>Elementti, josta togglea kontrolloidaan</span>
<div class="jj-toggle">
   Elementin sisältö...
</div>

Output:
Elementti, josta togglea kontrolloidaan
Elementin sisältö...

Esimerkki

Otsikkoon sidonta javascriptillä:

<span id="id">Elementti, josta togglea kontrolloidaan</span>
<div>
   Elementin sisältö...
</div>
<script>
     $("#id").jjNextToggleable();
</script>

Output:
Elementti, josta togglea kontrolloidaan
Elementin sisältö...
